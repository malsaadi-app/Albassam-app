## 🏗️ الهيكل التنظيمي

### الفروع التعليمية (مدارس)

**الهيكل:**
```
الفرع (Branch)
 └─ المرحلة (Stage: ابتدائي/متوسط/ثانوي/رياض)
     └─ القسم (Department: المواد الأدبية، اللغة الإنجليزية...)
         └─ الموظف (Employee)
```

**المرجعية المزدوجة للموظفين:**
- **مرجعية إدارية:** المرحلة (ابتدائي/متوسط/ثانوي)
- **مرجعية أكاديمية:** القسم/المادة (المواد الأدبية، الرياضيات...)

**مثال:**
```
معلم "أحمد"
├─ إداريًا: تابع للمرحلة الابتدائية
└─ أكاديميًا: تابع لقسم المواد الأدبية
```

```
الفرع (Branch)
```

## ⚙️ متطلبات قاعدة البيانات

### النماذج (Models)

#### Branch (الفرع)
```typescript
{
  id: string
  name: string
  nameEn: string
  type: 'EDUCATIONAL' | 'CORPORATE'
  stages?: Stage[]
  departments: Department[]
  employees: Employee[]
}
```

#### Stage (المرحلة) - للفروع التعليمية فقط
```typescript
{
  id: string
  name: string
  nameEn: string
  branchId: string
  branch: Branch
  departments: Department[]
  employees: Employee[]
}
```

#### Department (القسم)
```typescript
{
  id: string
  name: string
  nameEn: string
  type: 'ACADEMIC' | 'ADMINISTRATIVE'
  branchId: string
  branch: Branch
  stageId?: string  // optional, only for educational branches
  stage?: Stage
  employees: Employee[]
  supervisorId?: string  // for supervisory departments
  supervisor?: Employee
}
```

#### Employee (الموظف)
```typescript
{
  id: string
  nameAr: string
  nameEn: string
  nationalId: string
  nationality: string
  mobile: string
  email: string
  qualification: string
  specialization: string
  jobTitle: string
  
  // Organizational structure
  branchId: string
  branch: Branch
  stageId?: string  // administrative reference
  stage?: Stage
  departmentId: string  // academic reference
  department: Department
  
  // Supervisor (for teachers)
  supervisorId?: string
  supervisor?: Employee
  
  // Salary (from second file)
  baseSalary?: number
  housingAllowance?: number
  transportAllowance?: number
  otherAllowances?: number
  bankAccount?: string
  
  // System
  userId?: string
  user?: User
}
```
