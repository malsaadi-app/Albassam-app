# Albassam Group Platform — EN (US) Corporate Glossary

> أنت (محمد) المراجع النهائي للمصطلحات.

## HR
- طلب إجازة = Leave Request
- إجازة سنوية = Annual Leave
- إجازة اضطرارية = Emergency Leave
- إجازة بدون راتب = Unpaid Leave
- إجازة مرضية = Sick Leave
- طلب تعريف بالراتب = Salary Certificate Request
- تعريف راتب = Salary Certificate
- طلب بدل سكن = Housing Allowance Request
- بدل سكن = Housing Allowance
- طلب بدل تذاكر = Ticket Allowance Request
- بدل تذاكر = Ticket Allowance
- استقالة = Resignation Request
- تبرير = Justification
- تأخير = Late Arrival
- غياب = Absence
- انصراف مبكر = Early Departure

## Attendance
- حضور / تسجيل حضور = Check-in
- انصراف / تسجيل انصراف = Check-out
- سماح التأخير = Grace Period
- نطاق الموقع = Geofence
- خارج النطاق = Out of Geofence

## Procurement
- طلب شراء = Purchase Requisition (PR)
- أمر شراء = Purchase Order (PO)
- عرض سعر = Quotation
- مورد = Supplier

## Workflow
- سير عمل = Workflow
- اعتماد = Approval
- مراجعة = Review
- إحالة = Referral / Reassign
- إقفال = Close

## Payroll
- الرواتب = Payroll
- قفل الرواتب = Payroll Lock
- اعتماد الرواتب = Payroll Approval
- خصم = Deduction
