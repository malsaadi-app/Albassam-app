## 🛠️ أدوات التصميم

### **Option A: Figma** (مجاني، الأفضل للمبتدئين)

#### **الخطوات:**
1. سجل في Figma: https://figma.com (free)
2. أنشئ artboard جديد: 1024x1024
3. استخدم الأدوات:
   - Rectangle tool (R) للمربعات
   - Ellipse tool (O) للدوائر
   - Text tool (T) للنص
   - Pen tool (P) للأشكال المخصصة
4. أضف Gradient:
   - Select shape → Fill → Gradient
   - Color 1: #1D0B3E
   - Color 2: #2D1B4E
5. صدّر: File → Export → PNG → 1024x1024

**رابط مباشر:** https://www.figma.com/files/recent

---

### **Option B: Canva** (سهل جداً، للمبتدئين)

#### **الخطوات:**
1. اذهب إلى: https://www.canva.com
2. Create a design → Custom size → 1024x1024
3. استخدم:
   - Elements → Shapes (للدوائر والمربعات)
   - Text (للحروف)
   - Gradients (للخلفيات)
4. Template search: "app icon" (اختر واحد وعدّله)
5. Download → PNG → 1024x1024

**رابط مباشر:** https://www.canva.com/create/icons/

## 📐 دليل التصميم (Design Guidelines)

### **Do's ✅**

```
✅ استخدم مربع كامل (1024x1024)
✅ اجعل التصميم بسيط وواضح
✅ اختبر على أحجام صغيرة (60x60)
✅ استخدم ألوان قوية ومتباينة
✅ اجعل الأيقونة تعمل بدون نص
✅ استخدم PNG بجودة عالية
✅ احفظ نسخة احتياطية PSD/Figma
```

### **Don'ts ❌**

```
❌ لا تستخدم transparency (alpha channel)
❌ لا تضيف rounded corners (iOS يضيفها تلقائياً)
❌ لا تستخدم صور فوتوغرافية
❌ لا تضيف نص صغير (لن يُقرأ)
❌ لا تستخدم تفاصيل دقيقة جداً
❌ لا تنسخ أيقونات تطبيقات أخرى
❌ لا تستخدم شعار Apple أو أيقونات النظام
```

## 🚀 الخطوات التالية

### **بعد التصميم:**

1. **احفظ الملف الأساسي:**
   ```
   app-icon-master-1024.png
   ```

2. **استخدم مولد الأيقونات:**
   ```
   Icon Kitchen أو AppIcon.co
   → Upload 1024x1024
   → Download all sizes
   ```

3. **استخراج الملفات:**
   ```
   unzip ios-icons.zip
   ```

4. **نسخ إلى Xcode:**
   ```
   ios/App/App/Assets.xcassets/AppIcon.appiconset/
   ```

5. **Commit to Git:**
   ```
   git add ios/App/App/Assets.xcassets/
   git commit -m "Add app icons"
   git push
   ```

6. **Build & Verify:**
   ```
   GitHub Actions → Trigger build
   → Check IPA
   → Verify icon appears
   ```
