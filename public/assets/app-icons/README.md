### 1. Choose Your Favorite Icon

### 2. Generate All iOS Sizes

**Option A: Icon Kitchen (Free, Recommended)**
```
1. Go to: https://icon.kitchen/
2. Upload your chosen PNG (1024x1024)
3. Select: iOS
4. Click: Generate
5. Download: ZIP file with all 13 sizes
6. Extract to: ios/App/App/Assets.xcassets/AppIcon.appiconset/
```

**Option B: AppIcon.co (Free)**
```
1. Go to: https://www.appicon.co/
2. Upload PNG
3. Select: iPhone/iPad
4. Generate
5. Download iOS pack
```

**Option C: MakeAppIcon (Free)**
```
1. Go to: https://makeappicon.com/
2. Upload PNG
3. Generate icon set
4. Download Xcode bundle
```

### 3. Add to Xcode Project

```bash
# After generating all sizes:
cd /path/to/ios/App/App/Assets.xcassets/AppIcon.appiconset/

# Copy all PNG files from icon generator
# Make sure Contents.json is present
```

### 4. Commit to Git

```bash
git add ios/App/App/Assets.xcassets/AppIcon.appiconset/
git commit -m "feat: Add app icons (all iOS sizes) 🎨"
git push
```

### 5. Build & Verify

```bash
# Trigger GitHub Actions build
# OR local build:
cd ios/App
xcodebuild -scheme App archive
```
