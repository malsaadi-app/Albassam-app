## 🎨 App Icon Design Brief

### **Concept Options:**

**Option 1: School Building + HR Symbol**
```
Design Elements:
- Stylized school building (simple outline)
- HR/employee icon overlay
- Color: #1D0B3E (purple) + white
- Style: Flat, minimalist
- Mood: Professional, educational
```

**Option 2: Arabic Letter Badge**
```
Design Elements:
- Letter "ب" (Ba from Bassam) in center
- Circular or shield background
- Gradient: Purple to lighter purple
- Gold/yellow accent border
- Style: Modern, elegant
```

**Option 3: Abstract Workflow**
```
Design Elements:
- Flowing lines representing workflow
- Connected nodes (approval flow)
- Color: Purple gradient
- Style: Abstract, modern
- Mood: Efficient, connected
```

**Recommended:** Option 2 (Arabic letter) - Most recognizable and unique

### **Color Palette:**

**Primary:**
- Purple: #1D0B3E (brand color)
- Dark Purple: #2D1B4E
- Light Purple: #3D2B5E

**Accent:**
- Gold: #FFD700
- White: #FFFFFF

**Background:**
- Gradient: #1D0B3E → #2D1B4E

### **Typography:**
- NO text in icon (Apple guideline)
- Icon should be self-explanatory

### **Dos & Don'ts:**

**Do ✅**
- Keep it simple and bold
- Use high contrast
- Make it recognizable at small sizes
- Use square canvas (rounded corners added by iOS)
- Test at different sizes (small to large)

**Don't ❌**
- Use photos or screenshots
- Add alpha transparency
- Make corners rounded yourself
- Use too many colors (2-3 max)
- Add text or tiny details

- **Check In:** Tap "Check In" button on dashboard

- **View Salary:** Profile → Payslips

- **History:** Payslips → View All

### Q: App won't open / crashes
**A:** Try:
1. Restart your iPhone
2. Update to latest iOS version
3. Reinstall the app
4. Contact support if issue persists

### Q: Face ID not working
**A:** Ensure:
1. Face ID is enabled in iPhone Settings
2. You granted Face ID permission to app
3. Your face is clearly visible
4. Try disabling and re-enabling

### Q: Attendance not showing
**A:**
1. Pull down to refresh
2. Check internet connection
3. Verify you're logged in
4. Contact HR if data is missing

### Q: Can't download payslip PDF
**A:**
1. Check internet connection
2. Ensure sufficient storage space
3. Try again in a few minutes
4. Contact support if issue continues
