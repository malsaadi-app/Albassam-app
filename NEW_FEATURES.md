## 📱 How to Use

### Dashboard:
1. **Login** → Automatically redirects to dashboard
2. **View Stats** → See your tasks and requests at a glance
3. **Check Charts** → Understand your performance trends
4. **Quick Actions** → Use the floating bar at bottom for fast access
5. **Review Alerts** → Address urgent items highlighted in red/yellow

### HR Requests:
1. **Browse** → See all requests in beautiful cards
2. **Filter** → Use sidebar to find specific requests
3. **Select** → Click checkbox for bulk actions (HR/Admin)
4. **Process** → Approve/reject/return individually or in bulk
5. **Export** → Download data for reports
6. **View Details** → Click any card to see full information

### Request Details:
1. **Review** → Check the summary card for all info
2. **Timeline** → Follow the request journey
3. **Read Comments** → See all feedback and decisions
4. **Take Action** → Use the prominent buttons at bottom
5. **Add Comment** → Provide feedback when approving/rejecting

## 🆘 Need Help?

### Common Questions:

**Q: Where are my old requests?**  
A: All your data is preserved. Use filters to find older requests.

**Q: How do I approve multiple requests?**  
A: Select checkboxes, then click the bulk action button.

**Q: Can I export filtered data?**  
A: Yes! Export button exports only visible (filtered) requests.

**Q: Why can't I see some features?**  
A: Features are role-based. Contact your admin if you need access.

**Q: How do I undo an action?**  
A: Actions are final. Always review before confirming.
