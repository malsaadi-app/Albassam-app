# ورقة الغش السريعة - نظام سير العمل

**مرجع سريع لكل من يستخدم نظام الموافقات**

---

## 🎯 للموظفين - كيف أوافق على طلب؟

### خطوات سريعة (2 دقيقة):

```
1. افتح التطبيق
2. اذهب إلى: سير العمل → الموافقات
3. اضغط "مراجعة" على الطلب
4. اقرأ التفاصيل
5. اضغط "موافقة" أو "رفض"
6. أضف تعليق (اختياري)
7. أكد القرار
```

### الروابط المباشرة:

```
📊 لوحة الموافقات:
https://app.albassam-app.com/workflows/approvals

🔍 الطلبات المنتظرة:
https://app.albassam-app.com/workflows/approvals?status=PENDING

📈 الإحصائيات:
https://app.albassam-app.com/settings/workflow-builder/analytics
```

---

## ⚡ أوامر سريعة

### الموافقة:
```
✅ موافقة = اضغط الزر الأخضر
💬 أضف تعليق (اختياري)
✓ أكد
```

### الرفض:
```
❌ رفض = اضغط الزر الأحمر
💬 أضف سبب (مطلوب!)
✓ أكد
```

### الفلاتر:
```
🔍 البحث: اسم الموظف / رقم الطلب
📂 النوع: HR / مشتريات / صيانة / حضور
📊 الحالة: منتظرة / موافق / مرفوض
📅 التاريخ: 7 / 30 / 90 يوم / الكل
```

---

## 🚀 للمسؤولين - كيف أنشئ سير عمل؟

### خطوات سريعة (10 دقائق):

```
1. الإعدادات → سير العمل → منشئ سير العمل
2. اضغط "+ جديد"
3. املأ المعلومات الأساسية:
   - الاسم
   - الوصف
   - النوع
   - الفروع/المراحل
4. أضف الخطوات (2-4 خطوات):
   - اسم الخطوة
   - الموافق
   - الترتيب
   - التصعيد
5. فعّل الإشعارات
6. احفظ كمسودة
7. اختبر
8. انشر
```

---

## 📊 الإحصائيات الأساسية

### ماذا أتابع؟

```
1. الطلبات المنتظرة: هل فوق 10؟ → 🔴 عاجل
2. الطلبات المتأخرة: أي طلب > 3 أيام → ⚠️ تدخل فوري
3. معدل الرفض: > 20%؟ → 🔍 راجع الأسباب
4. متوسط الوقت: > 5 أيام؟ → 📈 حسّن السير
```

### أين أجد الإحصائيات؟

```
📍 الرئيسية → لوحة التحكم → بطاقات الإحصاء
📍 سير العمل → تحليلات
📍 الإعدادات → التقارير
```

## 🔔 الإشعارات

### الأنواع:

| الإشعار | المعنى | الإجراء |
|--------|--------|---------|
| 🔔 طلب جديد | وصل طلب ينتظر موافقتك | راجع الآن |
| ⚠️ طلب متأخر | طلب متأخر > 3 أيام | افتح فوراً |
| ✅ تمت الموافقة | شخص آخر وافق | للعلم فقط |
| ❌ تم الرفض | شخص آخر رفض | للعلم فقط |
| 📊 تذكير يومي | لديك X طلب منتظر | مراجعة يومية |

### كيف أفعّل الإشعارات؟

```
الإعدادات → الإشعارات → سير العمل → ✅ تفعيل الكل
```

### أنواع الموافقين:

```

SYSTEM_ROLE      → دور نظام (ADMIN)
```

## 🛠️ استكشاف الأخطاء السريع

### المشكلة: لا أرى الطلبات

**الحل:**
```
1. تحقق من الفلاتر → مسح الكل
2. تحقق من الصلاحيات → المدير
3. تحديث الصفحة (F5)
```

### المشكلة: الطلب عالق

**الحل:**
```
1. افتح الطلب
2. اضغط "..." → "تصعيد"
3. أو: تواصل مع الدعم
```

### المشكلة: لا أستلم إشعارات

**الحل:**
```
1. الإعدادات → الإشعارات → تفعيل
2. السماح بالإشعارات في المتصفح
3. تحقق من البريد (Spam)
```

### المشكلة: الموافقة لا تعمل

**الحل:**
```
1. تحديث الصفحة
2. تسجيل خروج ودخول
3. تواصل مع الدعم
```

## 💬 الأسئلة الشائعة (سريع)

**س: هل يمكنني التراجع عن موافقة؟**  
ج: لا. تواصل مع الإدارة.

**س: كم الوقت للموافقة؟**  
ج: يعتمد على النوع. عادة 1-5 أيام.

**س: من يستلم الإشعار؟**  
ج: الموافق المحدد في الخطوة الحالية.

**س: هل يمكنني تفويض شخص؟**  
ج: نعم. الإعدادات → التفويض.

**س: أين أجد تاريخ موافقاتي؟**  
ج: الموافقات → فلتر: تمت الموافقة.

**س: هل يعمل على الجوال؟**  
ج: نعم. تطبيق + متصفح.

**س: ماذا لو رفضت بالخطأ؟**  
ج: الموظف يعيد التقديم. تواصل معه.

**س: كيف أعرف من وافق؟**  
ج: افتح الطلب → تفاصيل الموافقات.
